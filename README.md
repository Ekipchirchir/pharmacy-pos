# Pharmacy POS System

A robust Point of Sale (POS) system built with Django for managing pharmacy operations, including inventory, sales, suppliers, and reports. This system is designed for Kenyan pharmacies with features like expiry tracking, simulated M-Pesa integration, and KRA eTIMS compliance.

## Features
- **Inventory Management**: Track drugs, batches, quantities, prices, and expiry dates.
- **Sales Processing**: Process sales with cart functionality and customer details.
- **Supplier Management**: Add and manage supplier information.
- **Expiry Alerts**: Highlight drugs nearing expiry within 90 days.
- **Reports**: View sales history.
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Admin Interface**: Manage data via Django admin.
- **Simulated Integrations**: M-Pesa and KRA eTIMS (placeholder for real APIs).
